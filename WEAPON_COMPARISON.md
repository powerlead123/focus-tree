# 武器系统对比：旧 vs 新

## 旧系统的问题 ❌

### 只有子弹数量的区别
```
1级：1发子弹
2级：1发子弹
3级：1发子弹
...
19级：15发子弹
20级：16发子弹
```

**问题**：
- 低级武器完全没用（谁会用1发的？）
- 高级武器只是"更多子弹"，没有质的变化
- 没有战术选择，只有"越多越好"
- 升级没有惊喜感

---

## 新系统的改进 ✅

### 每个武器有独特性能

#### 示例1：6级 vs 7级
```
6级 闪电链：
- 1发子弹
- 穿透能力 ⚡
- 高速
- 适合：对付成群敌人

7级 毒液喷射器：
- 3发子弹
- 散射能力 🌀
- 高射速
- 适合：覆盖大面积
```

**对比**：
- 6级虽然只有1发，但能穿透多个敌人
- 7级有3发，但不能穿透
- 各有优势，看场景选择

#### 示例2：17级 vs 18级
```
17级 量子激光：
- 2发子弹
- 穿透能力 ⚡
- 超高速（10）
- 攻击力7
- 适合：快速清理

18级 核弹发射器：
- 1发子弹
- 爆炸能力 💥
- 中速（5）
- 攻击力8
- 适合：范围轰炸
```

**对比**：
- 17级：速度快，穿透，适合精准打击
- 18级：单发慢，但爆炸范围大
- 不是"18级一定比17级强"

## 四大核心能力

### 1. 🎯 追踪（Homing）
```javascript
// 子弹自动追踪最近的敌人
if (bullet.homing) {
    // 寻找最近的敌人
    // 转向目标
    // 自动命中
}
```

**拥有武器**：8、10、14、16、19、20级

**优势**：
- 不需要瞄准
- 适合新手
- 对付快速移动敌人

### 2. 💥 爆炸（Explosive）
```javascript
// 击中后范围伤害
if (bullet.explosive) {
    // 对范围内所有敌人造成伤害
    enemies.forEach(e => {
        if (距离 < 爆炸半径) {
            e.hp -= 伤害 * 0.5;
        }
    });
}
```

**拥有武器**：11、14、15、18、19、20级

**优势**：
- 一发打多个
- 清屏效率高
- 适合成群敌人

### 3. ⚡ 穿透（Pierce）
```javascript
// 子弹穿透敌人
if (!bullet.pierce) {
    bullets.splice(bIndex, 1); // 普通子弹击中后消失
} else {
    // 穿透子弹继续飞行
}
```

**拥有武器**：6、9、12、13、16、17、20级

**优势**：
- 一发击中多个
- 对付排成一线的敌人
- 子弹利用率高

### 4. 🌀 散射（Spread）
```javascript
// 多发子弹散开发射
const spread = hasSpread ? 30 : 0;
for (let i = 0; i < count; i++) {
    const offsetX = (i - (count - 1) / 2) * spread;
    // 子弹分散发射
}
```

**拥有武器**：4、5、7、9、13、15、19级

**优势**：
- 覆盖面积大
- 容易命中
- 适合移动敌人

## 实战对比

### 场景：10个敌人排成一线

#### 旧系统
```
1级武器：需要10发子弹，10次射击
10级武器：需要2次射击（6发×2=12发）
20级武器：需要1次射击（16发）
```
**结论**：只是数量差异

#### 新系统
```
6级闪电链（穿透）：
- 只需1发子弹！
- 穿透所有10个敌人
- 最高效

15级烟花连爆（散射+爆炸）：
- 4发散射子弹
- 每发爆炸范围伤害
- 可能1次清空

20级创世神器（全能力）：
- 3发追踪穿透爆炸
- 自动瞄准+穿透+范围伤害
- 无脑碾压
```
**结论**：性能质的差异

## 升级的惊喜感

### 旧系统
```
5级 → 6级：2发 → 3发（多1发）
10级 → 11级：6发 → 7发（多1发）
15级 → 16级：11发 → 12发（多1发）
```
**感受**：无聊，只是多一发

### 新系统
```
5级 → 6级：
- 散射 → 穿透
- 2发 → 1发（看起来降级？）
- 但是！穿透能力让1发胜过10发！
- 惊喜！

7级 → 8级：
- 散射 → 追踪
- 3发 → 1发
- 但是！自动瞄准，百发百中！
- 惊喜！

10级 → 11级：
- 追踪 → 爆炸
- 2发 → 1发
- 但是！范围伤害，一发炸一片！
- 惊喜！
```
**感受**：每次升级都有新玩法
