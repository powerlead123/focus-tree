# 武器升级路线 - 逐级详解

## 📖 阅读指南

每个武器都会说明：
- 📊 基础属性
- ✨ 特殊能力
- 🆚 比上一级强在哪里
- 💡 使用技巧
- 🎯 适用场景

---

## 🔰 第一阶段：新手入门（1-3级）

### 1级：🔫 生锈手枪
```
攻击力：1 ⭐☆☆☆☆☆☆☆☆☆
射速：慢（500ms）
子弹数：1发
子弹速度：慢（4）
特殊能力：无
```

**武器特点**：
- 最基础的武器
- 单发慢速
- 威力最弱

**使用技巧**：
- 需要精准瞄准
- 一发一个，不要浪费
- 专注躲避敌人

**适用场景**：
- 熟悉游戏操作
- 学习敌人移动规律

---

### 2级：🏹 木制弓箭
```
攻击力：2 ⭐⭐☆☆☆☆☆☆☆☆
射速：中（450ms）
子弹数：1发
子弹速度：快（6）
特殊能力：无
```

**🆚 比1级强在哪里**：
1. ✅ 攻击力翻倍（1→2）
2. ✅ 子弹速度提升50%（4→6）
3. ✅ 射速稍快（500→450ms）

**为什么更强**：
- 子弹飞得更快，更容易命中移动的敌人
- 伤害翻倍，击杀效率提升
- 射速提升，输出频率更高

**使用技巧**：
- 利用高速子弹打移动敌人
- 不需要太多预判

**适用场景**：
- 对付快速移动的敌人
- 远距离狙击

---

### 3级：🔧 扳手投掷器
```
攻击力：3 ⭐⭐⭐☆☆☆☆☆☆☆
射速：中（400ms）
子弹数：1发
子弹速度：中（5）
特殊能力：无
```

**🆚 比2级强在哪里**：
1. ✅ 攻击力再提升50%（2→3）

**为什么更强**：

- 优先打血量低的敌人

**适用场景**：
- 对付单个强敌
- 需要快速清理低血量敌人

## ⚔️ 第二阶段：能力觉醒（4-5级）

### 4级：💙 冰霜枪
```
攻击力：2 ⭐⭐☆☆☆☆☆☆☆☆
射速：中（380ms）
子弹数：2发
子弹速度：中（5）
特殊能力：🌀 散射
```

**🆚 比3级强在哪里**：
1. ✅ 第一次获得特殊能力：散射！
2. ✅ 双发子弹，覆盖面积大
3. ✅ 射速更快（400→380ms）

**为什么更强**：
- **质的飞跃**：虽然单发伤害降低（3→2），但双发散射总伤害4点
- 散射让你不需要精准瞄准
- 覆盖面积大，容易命中移动敌人
- 可以同时打两个敌人

**使用技巧**：
- 站在敌人群中间，左右都能打到
- 不需要精准瞄准，散射会覆盖
- 对付左右移动的敌人特别有效

**适用场景**：
- 敌人左右分布
- 移动敌人
- 新手友好（不需要精准瞄准）

**里程碑**：🎉 第一个有特殊能力的武器！

## 🌟 第三阶段：能力进化（6-10级）

### 6级：⚡ 闪电链
```
攻击力：3 ⭐⭐⭐☆☆☆☆☆☆☆
射速：中（320ms）
子弹数：1发
子弹速度：快（8）
特殊能力：⚡ 穿透
```

**🆚 比5级强在哪里**：
1. ✅ 第一次获得穿透能力！
2. ✅ 子弹速度大幅提升（5→8）
3. ✅ 射速更快（350→320ms）

**为什么更强**：
- **革命性变化**：虽然从2发变成1发，但穿透能力让1发胜过10发！
- 一发可以击中排成一线的所有敌人
- 子弹不会消失，持续飞行
- 对付成群敌人效率极高

**使用技巧**：
- 站在敌人群的一侧，让子弹穿透整排
- 等敌人排成一线再射击
- 不要浪费在单个敌人上

**适用场景**：
- 敌人成群出现
- 敌人排成一线
- 需要高效清屏

**里程碑**：🎉 第一个穿透武器！战术深度大幅提升！

### 7级：💚 毒液喷射器
```
攻击力：2 ⭐⭐☆☆☆☆☆☆☆☆
射速：快（250ms）
子弹数：3发
子弹速度：慢（4）

```

**🆚 比6级强在哪里**：
1. ✅ 射速大幅提升（320→250ms）
2. ✅ 三发散射，覆盖面积更大
3. ✅ 弹幕压制流

**为什么更强**：
- 虽然失去了穿透，但获得了超高射速
- 三发散射形成弹幕，敌人难以躲避
- 虽然单发伤害低，但总DPS很高
- 适合不同的战术风格

**使用技巧**：
- 持续射击，形成弹幕
- 不需要精准，覆盖整个区域
- 适合压制移动敌人

**适用场景**：
- 敌人快速移动
- 需要持续火力压制
- 大面积覆盖

**战术选择**：
- 6级适合成群敌人（穿透）
- 7级适合移动敌人（弹幕）
- 根据场景选择
