# 专注小树 - 儿童作业专注力辅助工具

一个帮助孩子培养专注力的可视化网页应用，支持跨设备实时监控。

## ✨ 功能特点

### 核心功能
- 🌳 **小树林成长系统**：每5秒长出一棵新树，最终形成茂密森林
- ⏱️ **时间管理**：设定预期时间，实时倒计时，培养时间观念
- ☁️ **温和提醒**：走神时出现乌云，暂停长树但时间继续流逝
- 📱 **跨设备监控**：家长可在手机上实时监控和提醒（需配置 Firebase）
- 📊 **详细统计**：今日/本周/总成就，专注率分析，成果森林可视化
- 🎉 **效率分析**：结束后展示时间对比，让孩子看到专注的价值

### 新增功能
- 🏎️ **口算竞速**：游戏化口算练习，支持手动模式和答题模式
- 🛒 **背景商城**：用小树兑换金币，解锁精美背景图
- 🍃 **激励系统**：10棵小树=1金币，10金币=1块拼图，16块拼图=1张完整背景
- 🖼️ **个性化背景**：解锁的背景可用于专注力训练和口算竞速
- 🎈 **泡泡单词**：趣味记单词（外部链接）

## 🚀 快速开始

### 本地使用（无需配置）

1. 下载所有文件到本地
2. 用浏览器打开 `home.html`（首页）
3. 选择功能模块：
   - 专注力训练：`index.html`
   - 口算竞速：`race.html`
   - 背景商城：`shop.html`
4. 开始使用！

**注意**：本地模式下，所有页面需要在同一浏览器中打开以共享数据。

### 跨设备使用（推荐）

支持电脑和手机分别操作，需要配置 Firebase：

1. 按照 [Firebase 配置指南](FIREBASE_SETUP.md) 设置
2. 部署到 Cloudflare Pages 或任何静态托管服务
3. 孩子在电脑上打开，家长在手机上打开
4. 通过房间号配对连接

详细步骤请查看 [FIREBASE_SETUP.md](FIREBASE_SETUP.md)

## 📖 使用流程

### 孩子端
1. 输入任务名称（如"数学作业"）
2. 设定预期完成时间（10/20/30/45/60分钟）
3. 获得6位房间号（如果配置了 Firebase）
4. 开始专注，看着小树一棵棵长出来
5. 完成后查看成果和效率分析

### 家长端
1. 输入孩子显示的6位房间号
2. 实时查看孩子的专注状态
3. 发现走神时点击"走神"按钮
4. 孩子端会立即显示乌云提示

## 🎯 核心功能

### 1. 时间管理
- 开始前设定预期时间
- 实时倒计时显示
- 颜色提示：绿色（充足）→ 黄色（过半）→ 红色（紧张）→ 紫色（超时）

### 2. 成果可视化
- 每5秒长一棵树
- 走神时暂停长树
- 时间持续流逝，体现走神代价

### 3. 效率分析
- 预期时间 vs 实际用时
- 走神次数和暂停时长
- 计算如果不走神能节省多少时间
- 针对性鼓励语

### 4. 历史统计
- 今日/本周/总成就汇总
- 专注率分析（带表情反馈）
- 成果森林可视化展示
- 详细的任务记录列表

## 🛠️ 技术栈

- 纯前端实现，无需后端
- Firebase Realtime Database（可选，用于跨设备通信）
- 本地 localStorage 作为降级方案
- 响应式设计，支持手机和电脑

## ⚙️ 自定义设置

### 修改长树间隔
在 `child.js` 中修改：
```javascript
const TREE_GROW_INTERVAL = 5; // 改为你想要的秒数
```

### 修改乌云持续时间
在家长端页面直接调整（3-30秒）

### 修改预期时间选项
在 `index.html` 中修改时间按钮

## 💡 使用建议

1. **孩子端**：全屏显示效果更好，减少干扰
2. **家长端**：可以在手机上操作，方便监控
3. **时间设定**：根据孩子年龄和任务难度合理设定
4. **走神提醒**：温和提醒，不要过于频繁
5. **成果展示**：结束后和孩子一起查看统计，鼓励进步

## 🔒 隐私说明

- 所有数据存储在本地浏览器（localStorage）
- 如使用 Firebase，数据存储在你自己的 Firebase 项目中
- 不会上传任何数据到第三方服务器
- 房间号随机生成，24小时后自动过期（建议配置）
