# 阅读探险记 - 武器分配系统说明

## 评分公式

总分 = 套数得分 + 效率得分 + 难度得分 + 答案得分

### 各项得分规则

1. **完成套数** (10-40分)
   - 1套 = 10分
   - 2套 = 25分
   - 3套 = 40分

2. **做题效率** (4-20分)
   - 由家长根据孩子做题速度和专注度评定
   - 1星 = 4分
   - 2星 = 8分
   - 3星 = 12分
   - 4星 = 16分
   - 5星 = 20分

3. **题目难度** (6-30分)
   - 题目越难，得分越高
   - 1星 = 6分
   - 2星 = 12分
   - 3星 = 18分
   - 4星 = 24分
   - 5星 = 30分

4. **答案评分** (6-30分)
   - 根据答题正确率评定
   - 1星 = 6分
   - 2星 = 12分
   - 3星 = 18分
   - 4星 = 24分
   - 5星 = 30分

### 总分范围
- 最低分：10 + 4 + 6 + 6 = 26分
- 最高分：40 + 20 + 30 + 30 = 120分

## 武器向下兼容系统

获得高等级武器后，可以选择使用该等级及以下的所有武器：
- 获得10级武器 → 可选择1-10级任意武器
- 获得20级武器 → 可选择1-20级任意武器

这样孩子可以：
- 体验不同武器的特色
- 根据喜好选择武器
- 增加游戏的可玩性

## 敌人类型与技能

### 敌人类型
1. **普通型** (normal) - 标准血量和速度
2. **快速型** (fast) - 速度快，血量少
3. **坦克型** (tank) - 血量高，速度慢，显示血条
4. **成群型** (swarm) - 3-5个一起出现
5. **技能型** (debuff) - 携带特殊技能

### 敌人技能（Debuff）
1. **❄️ 冻结** (freeze) - 玩家原地冻结3秒，无法移动
2. **🐌 减速** (slow) - 玩家移动速度减半4秒
3. **🔄 反向** (reverse) - 玩家左右方向反转3秒
4. **😵 混乱** (confuse) - 玩家控制混乱2.5秒

### 敌人出现方式
- 70% 单个敌人
- 20% 成批出现（3-5个）
- 10% 大型Boss敌人

## 激励机制

- 做得越多（套数）→ 基础分越高 → 解锁更多武器
- 做得越快越专注（效率）→ 额外加分 → 更强武器
- 挑战越难的题（难度）→ 额外加分 → 更强武器
- 答得越准确（评分）→ 额外加分 → 更强武器

**目标**：激励孩子多做题、做难题、做得快、做得准！每次都能体验不同的武器和游戏乐趣！
