# Boss系统设计 ✅

## Boss出现时机

**时间点：** 游戏进行到1分30秒（5400帧）时

**出场效果：**
- 清除所有普通敌人
- 停止普通敌人生成
- 显示Boss警告鼓励语
- Boss从屏幕顶部缓慢入场

## Boss属性

### 基础属性
- **血量：** 500点（是普通敌人的50-100倍）
- **大小：** 120像素（3倍普通敌人）
- **移动速度：** 1-3像素/帧
- **击败奖励：** 1000分 + 5个随机道具

### 根据主题的Boss变化

| 主题 | Boss | 名称 | 颜色 |
|------|------|------|------|
| 🌌 space | 👾 | 阅读理解终极Boss | 紫色 |

| ❄️ ice | ❄️ | 冰封阅读巨兽 | 青色 |
| 🌳 forest | 🌳 | 森林阅读守护者 | 绿色 |

## Boss三阶段系统

### 第一阶段（血量 > 66%）
**特点：**
- 射击间隔：60帧（1秒）
- 普通攻击：单发环形弹幕
- 特殊攻击：3发追踪弹（每300帧/5秒）
- 移动模式：左右移动、圆形移动、追踪玩家

**攻击模式：**
```
普通攻击：1发子弹，360度环形
特殊攻击：3发追踪弹，间隔300ms
```

### 第二阶段（血量 33%-66%）
**特点：**
- 射击间隔：40帧（0.67秒）- 加快33%
- 普通攻击：双发环形弹幕
- 特殊攻击：8发扇形弹幕
- 移动速度提升

**攻击模式：**
```
普通攻击：2发子弹，360度环形
特殊攻击：8发扇形弹幕，覆盖90度范围
```

### 第三阶段（血量 < 33%）
**特点：**
- 射击间隔：30帧（0.5秒）- 加快50%
- 普通攻击：三发环形弹幕
- 特殊攻击：12发螺旋弹幕（带混乱debuff）
- 移动最快，攻击最密集

**攻击模式：**
```
普通攻击：3发子弹，360度环形
特殊攻击：12发螺旋弹幕，间隔50ms，带混乱效果
```

## Boss移动模式

### 模式0：左右移动
```javascript
boss.x += Math.sin(gameTime * 0.02) * 3;
```
- 正弦波左右摆动
- 幅度：±3像素

### 模式1：圆形移动
```javascript
boss.x = canvas.width / 2 + Math.cos(gameTime * 0.03) * 100;
boss.y = boss.targetY + Math.sin(gameTime * 0.03) * 30;
```
- 以屏幕中心为圆心
- 水平半径：100像素
- 垂直半径：30像素

### 模式2：追踪玩家
```javascript
if (boss.x < player.x) boss.x += 2;
if (boss.x > player.x) boss.x -= 2;
```
- 跟随玩家X坐标
- 速度：2像素/帧

**切换频率：** 每3秒（180帧）随机切换一次

## Boss攻击系统

### 普通攻击
**环形弹幕：**
- 第一阶段：1发
- 第二阶段：2发
- 第三阶段：3发
- 360度均匀分布
- 速度：4像素/帧
- 子弹大小：12像素

### 特殊攻击

```javascript
直接瞄准玩家当前位置

```

```javascript

12发子弹，螺旋发射
```

## Boss视觉效果

### 光环效果
- 脉动光环，颜色根据主题变化
- 半径：60-80像素
- 脉动频率：0.1弧度/帧

### 血条显示
- **位置：** Boss上方80像素
- **大小：** 300×20像素
- **颜色：**
  - 绿色（血量 > 66%）
  - 橙色（血量 33%-66%）
  - 红色（血量 < 33%）
- **显示内容：**
  - 血量数字：`500 / 500`
  - 阶段指示：`阶段 1/3`

### 名称显示
- Boss名称显示在血条上方
- 颜色与主题匹配
- 黑色描边，增强可读性

## Boss死亡效果

### 爆炸动画
```javascript
1. 主爆炸：Boss位置
2. 连环爆炸：10次，间隔100ms
3. 爆炸范围：±100像素随机
4. 爆炸大小：2-3倍
```

### 奖励掉落
- **分数：** +1000分
- **道具：** 5个随机道具
- **掉落间隔：** 200ms
- **掉落范围：** Boss位置±150像素

### 游戏恢复
- Boss消失
- 重启普通敌人生成
- 显示胜利鼓励语

## Boss碰撞检测

### 子弹击中Boss
- **判定范围：** 60像素
- **伤害：** 根据武器攻击力
- **穿透弹：** 击中后继续飞行
- **爆炸弹：** 造成范围伤害

### Boss撞击玩家
- **判定范围：** 80像素
- **伤害：** 20点（是普通敌人的4倍）
- **护盾/无敌：** 可以防御

## 战斗策略

### 推荐武器
- **高级武器（15级以上）：** 伤害高，快速击败
- **穿透武器：** 可以同时击中Boss和子弹
- **爆炸武器：** 范围伤害，清理弹幕

### 推荐道具
- **时停⏰：** 减慢Boss和子弹速度
- **冰冻❄️：** 冻结Boss，停止攻击
- **无敌✨：** 无视弹幕，专心输出
- **狂射🚀：** 射速加倍，快速削血
- **激光🔆：** 高伤害，清理弹幕

### 战斗技巧
1. **第一阶段：** 熟悉移动模式，躲避追踪弹
2. **第二阶段：** 注意扇形弹幕，走位到侧面
3. **第三阶段：** 螺旋弹幕最密集，优先使用道具

## 平衡性设计

### 难度曲线
- **血量：** 500点，需要持续输出
- **伤害：** 子弹5点，撞击20点
- **时长：** 预计30-60秒击败
- **奖励：** 1000分 + 5道具，非常丰厚

### 容错机制
- 玩家血量300点，可承受60次子弹攻击
- Boss出现前已经玩了1分30秒，应该有道具储备
- 血包掉落概率提高，确保生存能力
- 可以用子弹击落Boss子弹

```javascript
    spawnBoss();
```
