## 技术细节

### 移除的性能杀手

1. **粒子系统**
   ```javascript
   // 移除前：每个子弹维护多个粒子数组
   bullet.trail = [];
   bullet.particles = [];
   bullet.sparkles = [];
   bullet.sparks = [];
   bullet.nebula = [];
   bullet.tail = [];
   bullet.aura = [];
   
   // 移除后：只保留基本属性
   bullet.x, bullet.y, bullet.size, bullet.type
   ```

2. **shadowBlur**
   ```javascript
   // 移除前：每帧计算数百次
   ctx.shadowBlur = 20;
   ctx.shadowColor = '#ff0000';
   
   // 移除后：使用简单stroke
   ctx.strokeStyle = '#ff0000';
   ctx.lineWidth = 2;
   ```

3. **复杂渐变**
   ```javascript
   // 优化前：每帧创建多个渐变对象
   const gradient = ctx.createRadialGradient(...);
   gradient.addColorStop(0, ...);
   gradient.addColorStop(0.3, ...);
   gradient.addColorStop(0.6, ...);
   gradient.addColorStop(1, ...);
   
   // 优化后：简化为2-3个色阶
   const gradient = ctx.createRadialGradient(...);
   gradient.addColorStop(0, ...);
   gradient.addColorStop(1, ...);
   ```
