# 背景商城使用指南

## 🎯 激励系统概览

专注小树应用通过一个完整的激励系统，让孩子在学习中获得成就感：

```
专注力训练 → 获得小树 → 兑换金币 → 解锁拼图 → 获得背景图
```

## 📊 兑换规则

- **10棵小树** = 1个树叶金币 🍃
- **10个金币** = 解锁1块拼图
- **16块拼图** = 1张完整背景图 🖼️

## 🌳 如何获得小树

### 专注力训练
1. 打开首页，点击"专注力训练"
2. 输入任务名称和预期时间
3. 开始专注学习
4. **每5秒自动长出1棵小树**
5. 完成任务后，小树会自动保存

### 小树累积
- 小树会累积在你的账户中
- 在首页可以看到总小树数
- 可用小树数显示可以兑换的数量

## 🍃 如何兑换金币

### 方法一：首页兑换
1. 打开首页
2. 查看"可兑换提示"
3. 点击底部的"兑换金币"按钮
4. 选择兑换次数
5. 确认兑换

### 方法二：商城兑换
1. 进入背景商城
2. 点击右上角的金币图标
3. 选择兑换次数
4. 确认兑换

### 兑换示例
- 你有35棵小树
- 可以兑换3次（30棵小树）
- 获得3个金币
- 剩余5棵小树

## 🛒 如何使用商城

### 浏览背景图
1. 从首页点击"背景商城"
2. 查看所有可用的背景图
3. 每张背景图显示为4×4的拼图网格
4. 查看解锁进度（X/16）

### 解锁拼图块
1. 点击任意未解锁的拼图块（显示🔒）
2. 查看预览图
3. 确认需要10个金币
4. 点击"确认解锁"
5. 观看解锁动画

### 拼图特点
- **随机解锁**：每次解锁的位置是随机的
- **逐步显示**：解锁后可以看到该块的完整图像
- **进度追踪**：实时显示已解锁的块数

### 完成背景图
- 解锁全部16块拼图后
- 会显示庆祝动画 🎉
- 背景图标记为"已解锁"
- 可以在专注力训练和口算竞速中使用

## 🖼️ 如何使用背景图

### 在专注力训练中
1. 进入专注力训练设置
2. 找到"背景选择"选项
3. 选择已解锁的背景图
4. 开始训练时会应用该背景

### 在口算竞速中
1. 进入口算竞速设置
2. 找到"背景选择"选项
3. 选择已解锁的背景图
4. 比赛时会应用该背景

## 📈 进度追踪

### 首页统计
- **累计小树**：总共获得的小树数
- **金币数量**：当前拥有的金币
- **背景图**：已解锁的背景图数量（X/10）
- **连续天数**：连续学习的天数

### 商城进度
- 每张背景图显示解锁进度
- 进度条直观显示完成度
- 已完成的背景图显示"✓ 已解锁"

## ❓ 常见问题

### Q: 小树会过期吗？
A: 不会！小树会永久保存在你的账户中。

### Q: 金币可以退回小树吗？
A: 不可以。兑换是单向的，请谨慎兑换。

### Q: 解锁的拼图块可以取消吗？
A: 不可以。解锁后无法撤销，请确认后再解锁。

### Q: 可以跳过某些拼图块吗？
A: 不可以。必须解锁全部16块才能使用背景图。

### Q: 背景图可以重复解锁吗？
A: 不需要。一旦完全解锁，永久可用。

### Q: 数据会丢失吗？
A: 数据保存在浏览器本地，清除浏览器数据会丢失。建议定期备份。

### Q: 可以在不同设备上使用吗？
A: 目前数据保存在本地，不同设备需要分别积累。未来可能支持云同步。

### Q: 如何快速获得金币？
A: 坚持每天专注学习，积累小树，定期兑换金币。
