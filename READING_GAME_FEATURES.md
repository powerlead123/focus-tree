# 阅读探险记 - 游戏特性说明

## 🎮 核心升级

### 1. 武器系统（20种）
- **数量**：从10种增加到20种武器
- **分数区间**：每5分一个等级，更精细的区分
- **向下兼容**：获得高级武器后，可选择使用该等级及以下所有武器
- **武器选择界面**：录入完成后，进入武器选择界面，孩子可以自己选择喜欢的武器
- **游戏中切换**：游戏进行中可以随时切换武器，右上角有"换武器"按钮

### 2. 游戏区域限制
- **窄条移动区域**：飞机只能在屏幕中间60%的区域左右移动
- **左右边界**：各占屏幕20%，显示为暗色区域
- **边界线**：白色半透明线标记游戏区域
- **更像传统飞机大战**：窄条区域增加游戏挑战性和策略性

### 3. 道具系统（6种）

打得好会掉落奖励道具，连击5次以上开始掉落：

| 道具 | 图标 | 效果 | 持续时间 |
|-----|------|------|---------|
| 极速 | ⚡ | 移动速度翻倍 | 5秒 |
| 巨弹 | 💥 | 子弹体积变大，伤害更高 | 5秒 |
| 全屏 | 🌟 | 可以全屏移动（突破边界） | 5秒 |
| 护盾 | 🛡️ | 免疫伤害（不免疫debuff） | 6秒 |
| 多重 | 🎯 | 子弹数量翻倍 | 5秒 |
| 穿透 | 🔥 | 子弹可以穿透敌人 | 5秒 |

**道具特点**：
- 连击5次以上才会掉落
- 每8秒检查一次是否生成
- 最多同时存在3个道具
- 道具会旋转下落，有光晕效果
- 拾取后左上角显示道具图标和剩余时间
- 同类道具会刷新持续时间

### 4. 连击系统
- **连击判定**：2秒内连续击杀敌人
- **连击显示**：3连击以上在飞机下方显示"X COMBO!"
- **连击奖励**：5连击以上开始掉落道具
- **连击中断**：超过2秒未击杀，连击归零

### 5. 敌人多样性

#### 敌人类型（5种）
1. **普通型** (normal) - 标准属性
2. **快速型** (fast) - 速度快，血量少，难以击中
3. **坦克型** (tank) - 血量高，速度慢，显示血条
4. **成群型** (swarm) - 3-5个一起出现，形成弹幕
5. **技能型** (debuff) - 携带特殊技能，碰到会有负面效果

#### 敌人出现方式
- **70%** 单个敌人随机出现
- **20%** 成批出现（3-5个排成一排）
- **10%** 大型Boss敌人（血量特别高）

### 6. Debuff技能系统

玩家碰到带技能的敌人会受到负面效果：

| 技能 | 图标 | 效果 | 持续时间 |
|-----|------|------|---------|
| 冻结 | ❄️ | 原地冻结，无法移动 | 3秒 |
| 减速 | 🐌 | 移动速度减半 | 4秒 |
| 反向 | 🔄 | 左右方向反转 | 3秒 |
| 混乱 | 😵 | 控制混乱 | 2.5秒 |

**特点**：
- Debuff图标显示在飞机上方
- 冻结时飞机会显示冰冻特效
- 同类debuff会重置持续时间
- 护盾可以免疫伤害，但不能免疫debuff
- 增加游戏挑战性和趣味性

### 7. 视觉增强

- **游戏区域边界**：左右暗色区域+白色边界线
- **Boss血条**：大型敌人显示血条，实时显示剩余血量
- **爆炸效果**：根据敌人大小调整爆炸范围
- **Debuff提示**：玩家受到debuff时显示图标
- **双层爆炸**：爆炸有内外两层，更炫酷
- **护盾光环**：有护盾时显示蓝色光环
- **巨弹光晕**：巨弹有额外光晕效果
- **穿透标记**：穿透弹显示闪电标记
- **连击显示**：3连击以上显示金色COMBO文字
- **道具光晕**：道具有彩色光晕和旋转效果
- **道具计时器**：左上角显示激活的道具和剩余时间

## 🎯 游戏流程

1. **选择关卡** → 点击未完成的关卡
2. **家长录入** → 输入今日完成情况（套数、效率、难度、评分）
3. **武器预览** → 实时显示将获得的武器等级
4. **武器选择** → 从1级到获得等级的所有武器中选择
5. **开始游戏** → 使用选择的武器进行3分钟射击游戏
6. **游戏中切换** → 可以随时点击"换武器"按钮切换武器
7. **收集道具** → 连击5次以上会掉落道具，增强战斗力
8. **通关结算** → 显示得分、星级、奖励

## 💡 设计理念

### 激励机制
- **多样化武器**：20种武器让孩子有更多追求目标
- **向下兼容**：可以体验不同武器，增加可玩性
- **精细评分**：每5分一个等级，让进步更明显
- **自主选择**：孩子可以选择喜欢的武器，增加参与感
- **游戏中切换**：可以尝试不同武器的手感

### 游戏性
- **窄条区域**：更像传统飞机大战，增加挑战性
- **道具系统**：6种道具提供不同的战斗体验
- **连击系统**：鼓励连续击杀，打得好有奖励
- **敌人多样**：5种类型，不同属性，不会单调
- **技能系统**：debuff增加挑战性和趣味性
- **成批出现**：形成弹幕效果，更刺激
- **Boss战**：大型敌人带来成就感

### 平衡性
- **保证通关**：3分钟必定通关，不会挫败
- **难度适中**：debuff持续时间短，不会太难
- **道具门槛**：需要5连击才掉落，鼓励技术
- **护盾限制**：护盾不能免疫debuff，保持挑战
- **奖励明确**：表现越好，武器越强，体验越爽

## 🎨 道具效果组合

道具可以叠加，创造强大的组合效果：

- **极速 + 全屏**：超快速度全屏移动，躲避能力MAX
- **巨弹 + 多重**：大量巨型子弹，火力覆盖
- **穿透 + 多重**：多发穿透弹，清屏神器
- **护盾 + 极速**：无敌冲锋，横冲直撞
- **全屏 + 护盾**：全屏无敌，随意走位

## 📊 技术实现

- **Canvas渲染**：流畅的游戏画面
- **碰撞检测**：精确的碰撞判定
- **状态管理**：完善的debuff和道具系统
- **动态生成**：敌人和道具随机生成，每次不同
- **本地存储**：进度自动保存
- **实时切换**：游戏中可以切换武器
- **区域限制**：游戏区域动态计算

## 🎯 测试建议

1. **低分测试**：1套+全1星 = 26分 → 应获得1级武器
2. **中分测试**：2套+全3星 = 73分 → 应获得10级武器
3. **高分测试**：3套+全5星 = 120分 → 应获得20级武器
4. **武器选择**：确认可以选择获得等级及以下的所有武器
5. **游戏中切换**：点击"换武器"按钮，确认可以切换
6. **移动范围**：确认飞机只能在中间60%区域移动
7. **全屏道具**：拾取全屏道具后，确认可以全屏移动
8. **连击系统**：连续击杀，观察连击数和道具掉落
9. **道具效果**：测试每种道具的效果和持续时间
10. **Debuff测试**：碰到技能敌人，确认debuff生效
11. **护盾测试**：有护盾时碰敌人不扣血，但会受debuff
12. **敌人类型**：观察是否有单个、成批、Boss三种出现方式
